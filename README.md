# TechNova Enterprise LAN Project

## Overview
This project simulates a multi-floor enterprise network designed using Cisco Packet Tracer.

## Features
- VLAN segmentation (IT, HR, Finance, Sales)
- Inter-VLAN routing (Router-on-a-Stick)
- EtherChannel for redundancy and load balancing
- RSTP for loop prevention
- Subnetting using /28 networks

## How to Use
1. Download the `.pkt` file
2. Open with Cisco Packet Tracer
3. Explore configurations via CLI
